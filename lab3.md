## Задание 1. Корреляционная иммунность и эластичность

### Теоретические основы

Определение порядка корреляционной иммунности опирается на спектральные свойства функции (коэффициенты Уолша-Адамара):

> **Теорема:** Булева функция $f$ корреляционно иммунна порядка $k \Leftrightarrow \forall \vec u\in V_n, 1 \le ||\vec u|| \le k \implies W_f(\vec u) = 0$.
> 

Здесь:
* $||\vec u||$ — вес Хемминга вектора $\vec u$ (количество единиц в его двоичном представлении).
* $W_f(\vec u)$ — коэффициент Уолша-Адамара функции $f$ на векторе $\vec u$.
* Вектор $\vec u = (0, 0, \dots, 0)$ веса $0$ исключается (согласно следствию), так как равенство нулю $W_f(0)$ означает лишь сбалансированность функции.

**Алгоритм проверки:**
1. Проверяем коэффициенты $W_f$ при всех векторах веса $1$ (содержащих одну единицу, например, `000001`, `000010` и т.д.). Если все они равны $0$, то функция корреляционно иммунна порядка 1 ($k \ge 1$).
2. Если хотя бы один из них отличен от нуля, функция имеет порядок иммунности $k = 0$ (не является корреляционно иммунной).
3. При успешном прохождении шага 1 проверяем вектора веса 2, затем веса 3 и т.д.

*Эластичная* порядка $k$ функция — это сбалансированная булева функция, которая является корреляционно иммунной порядка $k$. Так как все наши функции сбалансированы (вес равен 32 при $2^n = 64$), порядок их эластичности в точности равен порядку их корреляционной иммунности.

### Результаты для нашего S-блока

Согласно расчетам, выполненным для шести координатных функций:
* **Функция $f_1$:** порядок корреляционной иммунности $k = 0$, эластичность порядка $0$.
* **Функция $f_2$:** порядок корреляционной иммунности $k = 0$, эластичность порядка $0$.
* **Функция $f_3$:** порядок корреляционной иммунности $k = 0$, эластичность порядка $0$.
* **Функция $f_4$:** порядок корреляционной иммунности $k = 0$, эластичность порядка $0$.

*Вывод:* Ни одна из координатных функций не является корреляционно иммунной даже первого порядка (у всех обнаружены ненулевые коэффициенты Уолша-Адамара для векторов входов веса 1).

## Задание 2. Спектр стат. структуры и наилучшие линейные приближения

Коэффициенты статистической структуры $\Delta_f(\vec u)$ показывают связь функции с линейными комбинациями входов $\langle \vec u, \vec x\rangle$:
$$P\{f(\vec x) = \langle \vec u, \vec x\rangle\} = \frac{1}{2} + \frac{\Delta_f(\vec u)}{2^n}$$
$$\Delta_f(\vec u) = 2^n \cdot \left(P - \frac{1}{2}\right) = 2^{n-1} W_f(\vec u)$$

Наилучшее линейное приближение для каждой функции определяется вектором $\vec u$, на котором абсолютное значение коэффициента стат. структуры $|\Delta_f(\vec u)|$ максимально.

Ниже приведены максимальные абсолютные значения коэффициентов стат. структуры и соответствующие им наилучшие линейные приближения (в виде векторов $\vec u = (u_0, u_1, u_2, u_3, u_4, u_5)$):

1. **Функция $f_1$:**
   * Максимальный коэффициент: $|\Delta_f| = 12.0$
   * Наилучшее приближение: $\vec u = [1, 1, 1, 1, 1, 0]$ (линейная функция $x_0 \oplus x_1 \oplus x_2 \oplus x_3 \oplus x_4$)
2. **Функция $f_2$:**
   * Максимальный коэффициент: $|\Delta_f| = 10.0$
   * Наилучшие приближения: $\vec u = [0, 1, 1, 0, 0, 1]$ ($x_1 \oplus x_2 \oplus x_5$) и $\vec u = [1, 0, 1, 0, 0, 0]$ ($x_0 \oplus x_2$)
3. **Функция $f_3$:**
   * Максимальный коэффициент: $|\Delta_f| = 14.0$

4. **Функция $f_4$:**
   * Максимальный коэффициент: $|\Delta_f| = 10.0$

## Задание 3. Автокорреляция и проверка на бент-свойства

### Теоретические основы

Коэффициент автокорреляции булевой функции на сдвиге $\vec u$ рассчитывается по формуле:
$$cor_f (\vec u) = \frac{1}{2^n} \sum_{x\in V_n} (-1)^{f(\vec x) \oplus f(\vec x \oplus \vec u)}$$

*Свойства:*
* При нулевом сдвиге автокорреляция всегда равна $1$: $cor_f(0) = 1$.
* Для бент-функций (когда $n$ четно) все коэффициенты автокорреляции при ненулевых сдвигах равны $0$ (функции с идеальными автокорреляционными свойствами).
* Альтернативный критерий бент-функции: все её нормализованные коэффициенты Уолша-Адамара равны по модулю и составляют $\pm 2^{-n/2}$ (в нашем случае для $n=6$ они должны быть равны $\pm 2^{-3} = \pm 0.125$, что соответствует коэффициентам стат. структуры $\Delta_f(\vec u) = \pm 4.0$ для всех векторов $\vec u$).

### Результаты для нашего S-блока

1. **Коэффициенты автокорреляции:**
   Векторы автокорреляции для всех шести координатных функций содержат широкий разброс значений в диапазоне от $-0.5$ до $0.5$ при ненулевых сдвигах (например, для $f_1$ значения автокорреляции включают в себя $-0.375$, $-0.25$, $-0.125$, $0.0$, $0.125$, $0.25$, $0.375$).
2. **Бент-свойства:**
   * Ни одна из шести координатных функций **не является бент-функцией**.
   * *Обоснование:* Спектры стат. структуры координатных функций не постоянны по модулю (значения варьируются, достигая $10.0$, $12.0$ и $14.0$, тогда как для бент-функций все коэффициенты стат. структуры на $V_6$ обязаны быть равными $\pm 4.0$).
