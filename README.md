# Probabilistic Methods in Cryptography

Репозиторий содержит материалы, скрипты и тексты для курсовой работы по дисциплине «Вероятностные методы в криптографии». В рамках курсовой работы проводится исследование криптографических свойств булевых функций и криптоанализ комбинирующего генератора псевдослучайных чисел.

## Структура репозитория

* **`first.py`** — базовый Python-скрипт, реализующий всю математическую логику (поиск свойств Жегалкина, Уолша-Адамара, запретов по дереву, а также симуляция LFSR и проведение корреляционной и алгебраической атак). Выводит результаты всех 4 лабораторных работ прямо в консоль.
* **`latex_generator.py`** — генератор отчета. Выполняет те же криптографические вычисления, но вместо вывода в консоль формирует готовый к компиляции файл `coursework.tex` по стандартам ГОСТ (шрифт 14pt, интервалы и т.д.).
* **`lab1.md` – `lab4.md`** — детальные теоретические разборы каждой лабораторной работы с формулировками заданий.
* **`lectures.md`** — оцифрованные лекции по курсу, собранные в единый справочный документ.

## Как запустить генератор курсовой (LaTeX)

Скрипт `latex_generator.py` поддерживает разные уровни детализации (verbosity). В зависимости от переданного флага он будет генерировать либо короткую справку, либо полноценную главу курсовой работы со всеми теоремами и определениями.

1. Убедитесь, что у вас установлен Python 3.
2. Запустите скрипт через командную строку:

   ```bash
   # Максимальный уровень (содержит всю теорию, леммы и определения)
   python latex_generator.py --verbosity max

   # Средний уровень (только краткие пояснения шагов)
   python latex_generator.py --verbosity mid

   # Минимальный уровень (строго задания и сухие ответы/цифры)
   python latex_generator.py --verbosity min
   ```

   **Генерация параметров:** По умолчанию скрипт использует `seed=42`, чтобы отчет был детерминированным и не менялся при каждом запуске (удобно для отладки). Если вы хотите сгенерировать уникальную работу (с новой подстановкой и гаммой), используйте флаг `--random` или укажите свой собственный seed:
   ```bash
   # Полностью случайная генерация (для финального уникального отчета)
   python latex_generator.py --random

   # Генерация с фиксированным seed
   python latex_generator.py --seed 123
   ```

3. После выполнения скрипта в папке появится файл `coursework.tex`.

## Как собрать PDF-отчет (через Overleaf)

Сгенерированный файл `coursework.tex` полностью настроен для сборки без дополнительных правок.
1. Зарегистрируйтесь / войдите на сайт [Overleaf](https://overleaf.com/).
2. Создайте новый пустой проект (Blank Project).
3. Скопируйте всё содержимое локального файла `coursework.tex` (Ctrl+A, Ctrl+C).
4. Замените содержимое `main.tex` в Overleaf на скопированный код (Ctrl+V).
5. В блоке `\begin{titlepage}` (в самом начале файла) впишите название вашего ВУЗа, факультет, номер группы и ФИО.
6. Нажмите кнопку **Recompile** (или используйте компилятор XeLaTeX/pdfLaTeX). Готовый PDF можно скачивать!
